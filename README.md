# SAYOKI BIRD DROP

サヨキバードを使ったスイカゲーム風の物理パズルです。GitHub Pagesにそのまま公開できます。

## 操作

- マウス移動 / ドラッグ：落下位置を調整
- クリック / タップ / スペース：落下
- ← →：落下位置を微調整
- P / Esc：ポーズ

## 実装機能

- Matter.js物理演算
- 同種接触による進化合体
- 進化図解表示
- タイトル画面
- ハイスコア保存（localStorage）
- ポーズ機能
- リトライボタン
- X共有ボタン
- BGM / SE

## 公開方法

1. このフォルダの中身をGitHubリポジトリにアップロード
2. Settings → Pages
3. Deploy from a branch
4. Branch: main / root を選択
